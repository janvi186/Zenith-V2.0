PROJECT ARCHITECTURE

Overview

This Intrusion Detection System (IDS) represents a fusion of Machine Learning and DBMS concepts. By utilizing high-dimensional network data, the system identifies malicious patterns in real-time, providing a robust defense layer against evolving cyber threats.

Core Stack

[Image of machine learning model development lifecycle]
  • Python & Flask (Logic Layer)
  • Random Forest (AI Engine)
  • SQLite (Persistent Storage)
  • CICIDS-2017 (Training Data)
  • HTML5 / CSS3 / JS (UI Layer)
  • Fetch API (Data Bridge)

Use Case

Designed for high-performance security monitoring in edge computing environments where immediate threat detection is critical for maintaining network integrity.
